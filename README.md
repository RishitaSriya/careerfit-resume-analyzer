# CareerFit – Resume Analyzer & Career Suggestions

**CareerFit** is a web-based tool that analyzes resumes and provides actionable feedback to improve career readiness. It helps users identify **strong areas, missing skills, and role fit**, and suggests **online courses** for skill improvement.

---

## **Features**

- **Resume Analysis:** Highlights keywords and identifies strong areas.
- **Role Fit Level:** Evaluates how well the resume matches selected job roles.
- **Missing Skills Detection:** Shows skills or sections absent in the resume.
- **Improvement Suggestions:** Actionable tips for enhancing your resume.
- **Course Recommendations:** Suggests online courses for missing skills.
- **Resume Completeness Check:** Ensures essential sections are present.
- **Interactive UI:** Modern design with colors, animations, popups, and progress bar.
- **PDF Report Generation:** Download a detailed report of your resume analysis.

---


---

## **Technologies Used**

- **Frontend:** HTML, CSS, JavaScript  
- **Libraries:** jsPDF (for PDF generation)  


---

## **Usage**

1. Open `careerfit.html` in a browser (or via GitHub Pages).  
2. Paste resume text in the input area.  
3. Select the desired job role.  
4. Click **Analyze Resume**.  
5. View **Role Fit Level**, **Strong Areas**, **Missing Skills**, **Suggestions**, and download **PDF report**.

---

## **Impact**

CareerFit helps students, freshers, and job seekers **quickly identify gaps in their resumes** and receive **actionable career guidance**. It’s ideal for hackathon demos and can be expanded into a full-fledged career guidance platform.
